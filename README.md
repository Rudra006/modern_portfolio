A fully animated portfolio showcasing 2+ years of full-stack work — Angular.js SaaS dashboards, Python microservices, streaming AI interfaces, and cloud-native deployments.

## 🚀 Key Features
- **✨ Advanced Animations**: Scroll-reveal transitions and interactive 3D tilt effects on project cards via Framer Motion.
- **🌗 Dynamic Theming**: Full light/dark mode with persistent local storage.
- **📱 Fully Responsive**: Mobile-first layout with an animated hamburger menu.
- **📨 Contact Form**: Direct-to-inbox form powered by FormSubmit.co.
- **💎 Glassmorphism UI**: Frosted glass aesthetic with gradient accents.
